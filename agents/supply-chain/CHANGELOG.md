# Changelog

All notable changes to the Supply Chain Security Agent will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-15

### Added
- Initial agent definition
- Ecosystem patterns for npm and PyPI
- Package health detection (abandonment, typosquatting)
- License detection patterns (SPDX)
- Vulnerability scoring guidance (CVSS, EPSS, CISA KEV)
- Prioritization framework
- Remediation guidance
- Role-specific prompts (security engineer, software engineer, engineering leader, auditor)
