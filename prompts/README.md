# Prompts

This directory contains prompt templates and examples organized by category.

## Structure

- `security/` - Security-related prompts
- `development/` - Development and coding prompts
- `analysis/` - Analysis and investigation prompts

## Usage

Browse the subdirectories to find prompt templates for your specific use case.
